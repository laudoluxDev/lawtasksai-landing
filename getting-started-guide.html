How It Works

1

Get Your License

Visit lawtasksai.com and purchase credits. You'll receive a license key via email (starts with lt_).

2

Install the Loader

The loader connects your AI assistant to LawTasksAI.

For OpenClaw users: Type this command in your chat:

/skill install lawtasksai

For Claude Desktop (MCP): Download the MCP server from lawtasksai.com/download

3

Start Using Tasks

Type your request in the same chat window where you installed the loader. Just describe what you need in plain English.

💡 Where do I type prompts? In the same place you normally chat with your AI — whether that's the OpenClaw terminal, Claude Desktop app, or web interface. LawTasksAI works through your existing AI assistant.

Example: Type this in your AI chat

"Calculate deadlines for a breach of contract case in Colorado. Complaint served February 17, 2026."

→ LawTasksAI automatically finds the right task and returns a formatted deadline table
