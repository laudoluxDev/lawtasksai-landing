System Requirements

LawTasksAI works best with agentic AI systems that can execute tools and manage files. Check compatibility with your preferred AI platform.

Compatibility Guide

LawTasksAI's functionality depends on your AI system's capabilities. Agentic systems provide the full experience, while basic chat interfaces have limitations.

✅ Full Compatibility

Agentic AI Systems

Complete LawTasksAI experience with autonomous skill execution.

  • ✅ OpenClaw — Best experience
  • ✅ Claude Desktop + MCP
  • ✅ Cursor IDE (with MCP)
  • ✅ Other MCP-compatible systems

Full Capabilities:

  • • Execute all 200+ skills automatically
  • • Save credentials and cache locally
  • • Download and manage documents
  • • Handle license validation
  • • Complete workflow automation
⚠️ Partial Compatibility

Function-Calling Systems

Can execute skills but requires manual setup and file handling.

  • ⚠️ ChatGPT Plus (with functions)
  • ⚠️ Claude via API (with tools)
  • ⚠️ Gemini (with function calling)

Can Do:

  • • Make API calls to LawTasksAI
  • • Process and analyze results
  • • Guide through workflows

Cannot Do:

  • • Save credentials persistently
  • • Download files automatically
  • • Manage local cache
❌ Limited Compatibility

Basic Chat Interfaces

Requires manual API interactions. Not recommended.

  • ❌ ChatGPT Web Interface
  • ❌ Claude Web Interface
  • ❌ Basic text-only LLMs

Major Limitations:

  • • No API calls
  • • No file operations
  • • No persistent state
  • • Manual workflow required

Workaround: Handle API calls manually and feed results to LLM for analysis.
