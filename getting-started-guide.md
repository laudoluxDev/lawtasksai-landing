# LawTasksAI Getting Started Guide

The current customer guide lives at:

https://lawtasksai.com/getting-started.html

Use `/install.html` for installation instructions. Use `/getting-started.html` for first-use guidance after LawTasksAI is connected.

Key points:

- Copy the install prompt from `https://lawtasksai.com/install.html`.
- Approve the browser connection with the email for the user's LawTasksAI account.
- Confirm the connection by asking: `Check my LawTasksAI credit balance.`
- Searches, category browsing, and balance checks do not use a credit.
- One credit is used when a licensed workflow is launched.
- LawTasksAI servers handle account approval, credits, catalog/search metadata, and licensed workflow delivery.
- LawTasksAI does not receive client prompts, uploaded documents, or task answers.
- The user's chosen AI assistant runs the workflow according to that assistant provider's privacy terms and settings.

LawTasksAI is software for attorneys and legal professionals. It is not a law firm and does not provide legal advice.
