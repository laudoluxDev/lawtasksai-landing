Start using LawTasksAI.

This page is for after LawTasksAI is installed. Use it to confirm the connection, run your first workflow, understand credits, and fix the most common first-use issues.

Install LawTasksAI Browse Task Library

First, confirm it is connected

Open the AI assistant where you installed LawTasksAI and start a new conversation. Then ask:

Check my LawTasksAI credit balance.
If your balance appears, the connection is working. Searches, category browsing, and balance checks do not use a credit.
If the assistant says it cannot find LawTasksAI, fully quit and reopen the AI assistant. Many MCP clients load tools only when the app starts.

Try a first workflow

You do not need to know the exact workflow name. Describe the legal task in ordinary language and ask LawTasksAI to search first.

Use LawTasksAI to search for a demand letter workflow for a personal injury matter.
Use LawTasksAI to show me available litigation categories.
Use LawTasksAI to help draft discovery requests for a breach of contract case.

LawTasksAI will return matching workflows. Pick the one you want to run. A credit is used when a licensed workflow is launched, not when you search or browse.

Credits and account approval

One credit runs one licensed workflow. Balance checks, category browsing, and search are free.

When installation or account approval is needed, use the email address for your LawTasksAI account. Manual license-key entry is only a backup path.

Privacy in plain English

LawTasksAI servers handle account approval, credits, catalog/search metadata, and licensed workflow delivery. LawTasksAI does not receive your client prompts, uploaded documents, or task answers.

After the licensed workflow is delivered, your chosen AI assistant runs the workflow according to that assistant provider's privacy terms and settings.

LawTasksAI is software for attorneys and legal professionals. It is not a law firm and does not provide legal advice. Review all output with professional judgment before using it.

If something is not working

The assistant says npm or npx is unavailable

Install Node.js first. Node.js is a standard helper app used by many modern tools, and it includes npm and npx automatically. Go to nodejs.org, download the recommended installer for your computer, run it, reopen your AI assistant, and paste the install prompt again.

LawTasksAI does not appear in your AI assistant

Fully quit and reopen the assistant. If it still does not appear, go back to the install page and paste the install prompt again so the assistant can re-check the setup.

The assistant asks for approval

Approve LawTasksAI when your AI assistant asks. In Claude Desktop, you may also need to set the LawTasksAI tools to always allow in the connector settings.

Use the install page. It has the official prompt to copy into your AI assistant.
