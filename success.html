Step 1 — Extract the ZIP

Double-click lawtasksai.zip to extract it. You'll see a folder called lawtasksai containing two things:

  • lawtasksai/ — the OpenClaw skill (SKILL.md + config)
  • lawtasksai/mcp/ — the MCP server for Claude Desktop, Cursor, Windsurf, and other MCP-compatible clients

💡 Your license key is pre-configured in both — no manual entry needed.

OpenClaw is a self-hosted AI assistant that runs on your Mac, Windows PC, or Linux server. If you don't have OpenClaw yet, download it here.

  1. Easy install (recommended): Open your OpenClaw chat and type:
    I just downloaded the LawTasksAI skill file to my Downloads folder. Please find it, unzip it if needed, and install it so I can use it.

    OpenClaw will find the folder and set everything up automatically.

  2. Manual install (optional): Copy the lawtasksai folder into your OpenClaw skills directory:
    • Mac/Linux: ~/.openclaw/skills/lawtasksai/
    • Windows: C:\Users\YOU\.openclaw\skills\lawtasksai\

    Then restart OpenClaw so it picks up the new skill.

  3. Start using! Just ask for any legal task in your OpenClaw chat.

The lawtasksai/mcp/ folder contains a Python MCP server that works with Claude Desktop, Cursor, Windsurf, and any other MCP stdio-compatible client.

Requirements

  • Python 3.10 or later — download here
  • Claude Desktop, Cursor, Windsurf, or another MCP-compatible client

Option A — Automatic (recommended)

Run the included installer — it detects all installed MCP clients and configures each one automatically:

cd ~/Downloads/lawtasksai/mcp
pip3 install -r requirements.txt
python3 install.py

Then restart your MCP client and look for the lawtasksai tools.

Option B — Manual (Claude Desktop)

1. Install dependencies:

cd ~/Downloads/lawtasksai/mcp
pip3 install -r requirements.txt

2. Open your Claude Desktop config file:

  • Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json

3. Add the following inside "mcpServers" (create the file if it doesn't exist):

{ "mcpServers": { "lawtasksai": { "command": "python3", "args": ["/full/path/to/lawtasksai/mcp/server.py"], "env": { "LAWTASKSAI_LICENSE_KEY": "YOUR_LICENSE_KEY" } } } }

Replace /full/path/to/ with the actual path to your extracted folder, and YOUR_LICENSE_KEY with the key shown above.

Option B — Manual (Cursor / Windsurf)

Open Settings → MCP Servers (or edit cline_mcp_settings.json) and add the same JSON block shown above.

The server is standard MCP stdio — any client that supports stdio transport will work.

4. Restart & verify

After restarting your client, you should see these tools available:

  • lawtasksai_search — find the right skill for any task
  • lawtasksai_execute — run a skill (costs 1 credit)
  • lawtasksai_balance — check remaining credits
  • lawtasksai_categories — browse all skill categories
